# library-template

Setup for building JavaScript libraries (using Rollup) with the following: Babel (7.x), Jest, Flow (0.75), ESLint and Prettier.

## Setup

```sh
yarn
```

Then do start and watch for changes

```
yarn run start
```

To build once

```
yarn run build
```

### Running tests

To run tests once

```sh
yarn run test
```

To run tests and watch for changes

```sh
yarn run test:watch
```
